<h1>bikumeter</h1>
<h3>App to track my return on investment from having bought myself a bike. Basically an app that brings in activities from Strava, filters them and sends the activities that correspond to commute rides to a spreadsheet where I calculate ROI and stuff.</h3>

<p>Stack</p>
<ol>
<li>Python</li>
<li>Google Spreadsheets</li>
<li>Strava API/Webhooks</li>
</ol>
